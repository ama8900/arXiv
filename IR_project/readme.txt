# 🧠 Intelligent Web Crawler & Analyzer

This project is a smart web crawler and analyzer built to extract, analyze, and visualize research paper metadata from an academic source. It features robust data collection, interactive visualizations, and a clean Streamlit dashboard interface.

## 🚀 Project Overview

* **Title**: Intelligent Web Crawler & Analyzer
* **Goal**: Crawl academic website(s), analyze crawlability, extract structured metadata, and visualize it through an interactive dashboard.

## 🛠️ Features

* ✅ Crawlability check via `robots.txt`
* ✅ Data extraction (titles, authors, subjects, links)
* ✅ Minimal JavaScript reliance (HTML parsing preferred)
* ✅ Clean visualizations (word clouds, bar charts, heatmaps)
* ✅ Streamlit dashboard interface
* ✅ Deployed on [Streamlit Cloud](https://your-streamlit-url)
* ✅ Code hosted on [GitHub](https://github.com/yourusername/your-repo)

## 📁 Tech Stack

* **Backend**: Python, BeautifulSoup, Pandas
* **Frontend**: Streamlit, Plotly, Matplotlib, WordCloud
* **Deployment**: Streamlit Cloud

## 📊 Dashboard Preview

The dashboard provides:

* Category & subject distribution
* Author statistics
* Word clouds of paper titles
* Individual paper explorer with metadata and links

## 📂 Setup Instructions

1. **Clone the repo**:

   ```bash
   git clone https://github.com/yourusername/your-repo.git
   cd your-repo
   ```

2. **Install dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

3. **Run locally**:

   ```bash
   streamlit run app.py
   ```

4. **Data Folder**: Add `.csv` files to the `data/` directory with paper metadata.
